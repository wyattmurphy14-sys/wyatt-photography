# Wyatt Photography Deploy

Domain target: `wyattphotography.com`

## GitHub Pages

1. Create a new GitHub repository.
2. Upload these files from this folder:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `CNAME`
   - `assets/outdoor-booking.png`
3. In the repository, open Settings > Pages.
4. Set the source to deploy from the main branch.
5. Point the domain DNS to GitHub Pages for `wyattphotography.com`.

The file `CNAME` already contains `wyattphotography.com`.
